Fonctionnement: 
La page d'accueil(home) permet d'effectuer une convertion d'une devise a une autre. 

Selon les donnes renvoyees par l'api il faut fixer la devise de conversion.

Pour l'instant elle est sur l'EURO. Nous pouvons la faire varier dans le code source et cela marche parfaitement.

Mais nous reussirons a la faire varier automatiquement dans les prochains jours.

La page taux permet de connaitre la valeur d'une ou de plusieurs monnaies en fonction d'une monnaie de base
